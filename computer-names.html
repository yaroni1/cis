Computer Naming Convention - University of Haifa City Campus

There are 3 types of computer stations with corresponding naming conventions:

1. Desktop Computers in Faculty/Staff Offices

Format: [prefix]-[username]-[building][room]

Components:

  • Prefix (up to 3 letters): cis, is, or cs
  • Username: The user's univ email username (e.g., iborodko)
  • Building code: Single letter abbreviation
    • a = Amir building
    • d = Dylan building
  • Room number: The office room number (e.g., 213)
  • Separator: Hyphen (-) between each field

Example: cs-iborodko-a213

2. Laptops or Home Desktop Computers

Format: [prefix]-[username]-[inventory]

Components:

  • Prefix (up to 3 letters): cis, is, or cs
  • Username: The user's univ email username (e.g., iborodko)
  • Inventory number: 6-digit computer inventory number (e.g., 313266)
  • Separator: Hyphen (-) between each field

Example: cs-iborodko-313266

3. Public Station Computers (Classrooms, etc.)

Format: cis-[inventory]-[building][room]

Components:

  • Prefix: Always cis
  • Inventory number: 6-digit computer inventory number (e.g., 313266)
  • Building code: Single letter abbreviation
    • a = Amir building
    • d = Dylan building
    • h = Hachsharat Hayishuv building
  • Room number: The classroom/room number (e.g., 730)
  • Separator: Hyphen (-) between each field

Example: cis-313266-h730
